## Packages Explained

### Package java.io

Package java.io provides for system input and output through data streams, serialization and the file system.

Topics covered in this tutorial include:

* Byte streams - handle I/O of raw binary data.
* Character streams - handle I/O of character data, automatically handling translation to and from the local character sets.
* Buffered streams - optimize input and output by reducing the number of calls to the native API.
* Formatting - allows a program to write formatted text.


* Data streams - handle binary I/O of primitive data types and String values.
* Object streams - handle binary I/O of objects.

### Package java.nio.file

Package java.nio.file defines interfaces and classes for the Java virtual machine to access files, attributes, and file systems.

* java nio.file package - defines classes to access files and file systems.
* java.nio.file.attribute package - defines the API to access file and file system attributes.
* java.nio.file.spi package - is used by service provide implementations wishing to extend the platform default provider, or to construct other provider implementations.
